# Creativefurnituresolution

This repository contains a premium single-page website for Creative Furniture Solutions.

Files added:
- index.html — homepage and product gallery
- styles.css — site styles
- script.js — product rendering and simple cart interactions

To run locally:
1. Open `index.html` in your browser (no server required) or run a simple static server:

```bash
# Python 3
python3 -m http.server 8000

# then visit http://localhost:8000
```

Products use Unsplash images as placeholders. Replace with your own images and real checkout integration when ready.
